# Agent Instructions

## Environment

Always use the `imbue` conda environment when running or testing code in this project.

```bash
conda activate imbue
```

All commands (running the server, tests, installing packages) should be executed inside this environment.

## Tools

For mngr and vet usage, refer to [docs/mngr.md](docs/mngr.md) and [docs/vet.md](docs/vet.md).
